# Scrapsama

Scraper pour indexer des séries dans une base de données.

## Installation

```bash
git clone https://github.com/Oriloo/scrapsama.git
cd scrapsama
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install -e .
```

## Utilisation

```bash
source scrapsama-env/bin/activate
scrapsama-index
scrapsama-index-all
scrapsama-index-new
```
